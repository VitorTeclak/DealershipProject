🚗 Dealership Project
📖 Sobre o projeto

O Dealership Project é um sistema desenvolvido em Java com integração a banco de dados relacional, com o objetivo de simular o gerenciamento de uma concessionária de veículos.

A aplicação foi construída com foco na prática de conceitos fundamentais de backend, incluindo persistência de dados, organização em camadas e aplicação de boas práticas de desenvolvimento.

⚙️ Funcionalidades
Cadastro e gerenciamento de veículos
Cadastro e gerenciamento de clientes
Persistência de dados em banco de dados
Operações de consulta, inserção, atualização e remoção (CRUD)
💻 Tecnologias utilizadas
Java
JDBC (Java Database Connectivity)
Banco de dados relacional (SQL)
Programação Orientada a Objetos (POO)
🧠 Arquitetura e conceitos aplicados
Separação de responsabilidades (camadas)
Organização do código para facilitar manutenção
Modelagem de entidades do domínio (veículos, clientes, etc.)
Integração entre aplicação e banco de dados
Estruturação de operações de persistência (CRUD)
🎯 Objetivo

O principal objetivo do projeto é consolidar conhecimentos em desenvolvimento backend com Java, incluindo integração com banco de dados e construção de uma base sólida para evolução futura.
